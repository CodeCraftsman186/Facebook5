# Facebook广告测试周期全指南
![替代文字](93a3c1560684534eb17a3aac0182183.jpg)
## 一、基础测试周期框架
### 1. 冷启动阶段
- **最短有效测试期**：3-7天
  - 前24小时：系统学习期（避免频繁调整）
  - 第2-3天：初步数据收集
  - 第4-7天：优化淘汰期

- **关键指标阈值**：
  - CPC > 行业均值1.5倍 → 立即优化
  - CTR < 1% → 素材迭代
  - 转化率连续2天低于1% → 受众调整

### 2. 动态优化周期
- **Meta算法特性**：7天价值优化周期需完整跑完
- **优质广告组**：持续投放14-21天
- **衰退广告组**：3天无转化即暂停

## 二、分层测试时间矩阵
| 测试维度       | 建议时长  | 评估标准                     |
|----------------|----------|----------------------------|
| 受众定位       | 5-7天    | 对比3组相似受众CPM差异      |
| 创意组合       | 3-5天    | 同一受众下CTR差异≥20%       |
| 落地页优化     | 7-10天   | 跳出率降低15%以上           |
| 出价策略       | 全程监控  | 每24小时检查CPA波动         |
| 版位测试       | 2-3天    | Instagram与FB流量占比       |

## 三、智能优化节奏
- **实时监控**：AI系统每5分钟扫描异常波动
- **日粒度优化**：
  - 早8点：调整预算分配
  - 下午3点：更换衰退素材
  - 晚10点：扩量优质广告组
- **周粒度调整**：重置表现平庸的广告组

## 四、行业特定期限
- **电商类**：
  - 旺季测试压缩至3天
  - 动态素材每12小时刷新
- **SaaS类**：
  - 需要14天完整周期
  - 重点关注7日留存率
- **游戏类**：
  - 72小时判定LTV潜力
  - 每8小时更新创意素材

## 五、关键决策时间点
1. **24小时**：淘汰CTR垫底20%的素材
2. **72小时**：关闭CPA超均值2倍的组
3. **7天**：确定主力投放方向
4. **14天**：完成最终资源倾斜
